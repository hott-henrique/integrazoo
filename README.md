# INTEGRAZOO

Integrazoo is an app associated with the INTEGRAZOO university extension program at the Federal University of São João del-Rei. The objective is to facilitate the carrying out of zootechnical records for small producers. Initially working only with dairy producers, the project has great potential to bring benefits to the management of various animals.

## Features
- [x] Record milk production, per cow or herd.
- [ ] Keep track of reproduction and related treatments.
- [ ] Track medication use in individuals.
- [ ] Inform future management decisions based on past data.
